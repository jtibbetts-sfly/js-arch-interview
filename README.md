# js-arch-interview
High level js interview project

# Requirements
* node (14.3.1)
* yarn
* git

# Step 1 clone the repo

# Step 2 install deps
`yarn install`

# Step 2 start server and client

## To start (server)
`cd ./server`
`yarn start`

## To start (client)
`cd ./client`
`yarn start`
